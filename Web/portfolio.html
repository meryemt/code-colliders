Web Challenge

Here is your starting point for building your portfolio

Instructions

You will be creating a team portfolio which will be written using HTML, CSS, Javascript and (possibly) JQuery

You need to work as a team to add the following as a minimum: -

  • Text
  • Lists
  • Text Formatting e.g. different fonts, colours
  • Images
  • Links to other pages

To score extra point you will need to add some of or all of the following: -

  • Form
  • Hovers
  • Buttons
  • Login pages
  • JavaScript Function of your choice

Rules

  1. Each team member must participate and write a minimum of one page in the portfolio
  2. Resources should be used before asking mentors
  3. Points will be awarded for creativing on top of the above
